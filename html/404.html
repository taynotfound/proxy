Webserver Not Found
What happened?
You've requested a domain name which points to an IP address
that is part of the Waya network but no associated webserver could be found.

How to resolve?
Connect to the server and get started in /root/proxy/src/hosts.js.
The documentation can be found on GitHub at github.com/Luna-devv/proxy.
If you don't know what you're doing at this point, please contact Luna.
